# Currency Exchange Rate Data Cleaning & Analysis Using Excel

## Project Overview

This project focuses on cleaning and preparing historical currency exchange rate data using Microsoft Excel. The dataset contains exchange rates for multiple countries across different years, with several missing values that require systematic treatment before analysis.

The objective is to improve data quality by applying interpolation techniques, forward filling, backward filling, and data validation methods to create a complete and analysis-ready dataset.

## Business Problem

Financial and economic datasets often contain incomplete records due to missing observations, reporting delays, or data collection issues. Missing exchange rate values can impact trend analysis, forecasting, and reporting accuracy.

This project demonstrates practical data cleaning techniques used to restore missing values while preserving data consistency and reliability.

## Project Tasks

### Data Cleaning

* Created a backup copy of the original dataset.
* Identified missing values using Conditional Formatting.
* Applied Linear Interpolation for missing values between known exchange rates.
* Performed Forward Filling for missing values at the end of a series.
* Performed Backward Filling for missing values at the beginning of a series.
* Filled rows containing no valid data with zero values.
* Rounded all exchange rates to two decimal places.

### Reporting & Analysis

Used Excel functions to answer business questions in the Solution sheet without manual calculations.

## Methodology

### 1. Missing Value Identification

Used Conditional Formatting to highlight blank cells and identify data quality issues.

### 2. Linear Interpolation

Calculated missing values occurring between two known exchange rates using interpolation formulas.

### 3. Forward Fill

Propagated the most recent known exchange rate to subsequent missing values.

### 4. Backward Fill

Filled leading missing values using the first available exchange rate.

### 5. Data Validation

Verified calculated values and maintained consistency across all countries and years.
